# 在线硬件测试工具

一个基于 Vue 3 + Tailwind CSS 开发的在线硬件测试工具，提供键盘、鼠标、屏幕、麦克风和摄像头的检测功能。

## 项目概述

本项目是一个现代化的在线硬件测试工具，旨在帮助用户快速检测和验证计算机硬件的功能状态。通过直观的界面和实时反馈，用户可以轻松测试键盘按键、鼠标按钮、屏幕显示、麦克风输入和摄像头输出等硬件设备。

## 功能特性

- **键盘测试** - 实时检测104键键盘按键状态，支持按键高亮显示和测试记录
- **鼠标测试** - 检测鼠标左键、右键、中键、侧键状态，支持滚轮测试
- **屏幕检测** - 显示屏幕信息，支持颜色测试、坏点检测和响应时间测试
- **麦克风测试** - 检测麦克风状态，实时显示音量级别和音频波形
- **摄像头测试** - 检测摄像头状态，实时显示视频画面

## 技术栈

- **Vue 3** (Composition API) - 渐进式 JavaScript 框架
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Vite** - 下一代前端构建工具
- **Web Audio API** - 音频处理
- **MediaDevices API** - 媒体设备访问
- **Web Fullscreen API** - 全屏模式支持

## 使用说明

### 键盘测试

1. 点击左侧导航栏的"键盘测试"
2. 在物理键盘上按下任意按键
3. 页面上对应的虚拟键位会高亮显示（绿色）
4. 已测试的键位会保持浅色高亮
5. 点击右下角"重置"按钮清空测试记录

### 鼠标测试

1. 点击左侧导航栏的"鼠标测试"
2. 在鼠标模型区域点击不同的按钮
3. 实时查看各按钮状态和滚轮数值
4. 支持左键、右键、中键、前进键、后退键检测

### 屏幕检测

1. 点击左侧导航栏的"屏幕检测"
2. 查看屏幕基本信息（分辨率、尺寸、颜色深度等）
3. 点击颜色方块进入全屏颜色测试
4. 在全屏模式下：
   - 点击鼠标左键切换颜色
   - 滚动鼠标滚轮切换颜色
   - 按键盘左右方向键切换颜色
   - 按 ESC 键退出全屏

### 麦克风测试

1. 点击左侧导航栏的"麦克风测试"
2. 点击"开始测试"按钮
3. 对着麦克风说话或发出声音
4. 观察音量级别和波形变化
5. 点击"停止测试"结束测试

### 摄像头测试

1. 点击左侧导航栏的"摄像头测试"
2. 点击"开始测试"按钮
3. 允许浏览器访问摄像头权限
4. 观察视频画面是否清晰流畅
5. 查看摄像头信息和测试时长
6. 点击"停止测试"结束测试

## 技术实现

### 键盘测试实现

- 使用 `keydown` 和 `keyup` 事件监听键盘输入
- 使用 `event.preventDefault()` 阻止浏览器默认快捷键行为
- 使用 Vue 3 的响应式系统跟踪按键状态
- 使用 Tailwind CSS 实现按键高亮和动画效果

### 鼠标测试实现

- 使用 `mousedown`、`mouseup` 和 `wheel` 事件监听鼠标操作
- 支持检测鼠标左键、右键、中键和侧键（前进/后退）
- 实时显示滚轮滚动数值
- 使用罗技鼠标模型作为视觉参考

### 屏幕检测实现

- 使用 `window.screen` API 获取屏幕信息
- 使用 `document.documentElement.requestFullscreen()` 实现全屏模式
- 支持多种颜色测试和坏点检测
- 支持键盘和鼠标控制颜色切换

### 麦克风测试实现

- 使用 `navigator.mediaDevices.getUserMedia()` 获取麦克风权限
- 使用 `AudioContext` API 处理音频数据
- 实时显示音频波形和音量级别
- 支持开始/停止测试控制

### 摄像头测试实现

- 使用 `navigator.mediaDevices.getUserMedia()` 获取摄像头权限
- 使用 `<video>` 元素显示摄像头画面
- 实时显示摄像头信息和测试时长
- 支持开始/停止测试控制

## 界面设计

### 设计风格

- 现代科技感主题，采用深色背景和玻璃态效果
- 响应式布局，适配不同屏幕分辨率
- 流畅的动画和过渡效果
- 统一的视觉风格和配色方案

### 布局结构

- 左侧可折叠导航栏，包含测试功能入口
- 右侧主内容区域，显示测试界面
- 响应式设计，在小屏幕设备上自动调整布局

## 浏览器兼容性

- Chrome >= 90
- Firefox >= 88
- Edge >= 90
- Safari >= 14

## 注意事项

1. 麦克风测试和摄像头测试需要浏览器权限授权
2. 建议使用最新版本的浏览器以获得最佳体验
3. 屏幕检测功能在全屏模式下效果最佳
4. 键盘测试会屏蔽浏览器的默认快捷键
